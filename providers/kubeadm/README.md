# kubeadm

```
.
├── .kube     // Control-Plane のマニフェストを管理
├── boot      // Raspberry Pi において Linux Kernel に渡すパラメータを管理
├── etc       // Control-Plane / Data-Plane ホストマシンの `/etc` ディレクトリの構成情報を管理
├── hosts     // Raspberry Pi ホストマシンの設定ファイルを管理
├── logs      // Control-Plane / Data-Plane デプロイ時のログ情報を管理
└── scripts   // kubeadm に関連するスクリプトを管理
```
